task 1. В ходе данной задачи был реализован следующий макет с использованием HTML и CSS:

Для создания макета были выполнены следующие действия:

  • Создана веб-страница с заголовком и двумя секциями для размещения контента.
  • В первой секции (.section_1) были размещены карточки, которые автоматически располагаются в рядах благодаря применению CSS Grid.
  • Каждая карточка (.card_task1) содержит заголовок, текст и кнопку для интерактивности.

Создание макета включало следующие этапы:

  • Определение общих переменных для цветов и шрифтов, обеспечивая единый стиль дизайна.
  • Применение сброса стилей с использованием Normalize.css, чтобы обеспечить единообразный вид на различных браузерах.
  • Задание стилей для заголовков, карточек и кнопок, включая цвета, размеры шрифтов и выравнивание.
  • Использование медиазапросов для адаптивного дизайна, обеспечивая оптимальное отображение на различных устройствах.

Расположение карточек и их структура были реализованы следующим образом:

  • В контейнере .cards был применен CSS Grid с использованием свойства grid-template-columns, обеспечивая автоматическое размещение карточек в рядах и столбцах.
  • Каждая карточка (.card_task1) имеет установленную минимальную ширину и заданное свойство flex-basis для равномерного распределения карточек в рядах, даже если количество элементов в ряду ограничено шириной экрана.

В результате была создана веб-страница с адаптивным макетом, содержащим карточки, которые могут располагаться в нескольких рядах в зависимости от размеров экрана пользователя.

task 2. В ходе данной задачи было выполнено следующее:

Для выполнения задачи были предприняты следующие шаги:

  • Использован jQuery для манипуляций с DOM элементами и обработки данных.
  • Создана функция sortBlocks() для сортировки блоков.
  • Внутри функции sortBlocks() используется $(document).ready() для обеспечения выполнения кода после загрузки DOM.
  • Используя селектор $('.section_2 .card[data-attribute]'), найдены все карточки с атрибутом data-attribute.
  • Сортировка карточек проведена с использованием метода sort() и сравнения их ширин с помощью parseInt($(element).css('width')).
  • Карточки с data-атрибутом были отсоединены от DOM методом detach() и затем вставлены обратно в контейнер .section_2.
  • Карточка без data-атрибута также отсоединена методом detach() и перемещена в конец контейнера .section_2.
  • Установлена высота всех карточек во второй секции равной высоте самой высокой карточки с дополнительным отступом (30 пикселей).

task 2.1 В ходе данной задачи была реализована карусель изображений с помощью чистого JavaScript:

Для создания карусели были выполнены следующие действия:

  • Создан контейнер для карусели с классом .carousel и кнопки "Предыдущий" и "Следующий" с классами .prev-button и .next-button.
  • Создан массив путей к изображениям, которые были взяты с официального сайта meetorra.ru из раздела "Наши клиенты".
  • На основе массива изображений созданы слайды с классом .slide и добавлены в контейнер карусели.
  • Общее количество слайдов было определено как длина массива изображений.
  • Установлено начальное смещение для карусели с помощью свойства transform: translateX.
  • Написаны функции для плавного перехода к следующему и предыдущему слайду с учетом возможности зацикливания карусели.
  • Задан интервал для автоматической смены слайдов с помощью setInterval.
  • Добавлены обработчики событий для кнопок "Предыдущий" и "Следующий", которые вызывают соответствующие функции перехода.
  • Реализована плавная анимация перехода между слайдами с помощью CSS свойства transition и transform.
  • Кроме того, карусель адаптируется при уменьшении экрана, уменьшая отображаемое количество изображений для улучшения пользовательского опыта на мобильных устройствах.
